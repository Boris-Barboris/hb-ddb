    Function PGConnection.execute

    Executes a non query command, i.e. query which doesn't return any rows. Commonly used with data manipulation commands, such as INSERT, UPDATE and DELETE.

    Prototype

    ulong execute(
      string query
    ) @safe;

    Parameters

    NameDescription
    query query string to execute

    Examples

    auto deletedRows = conn.execute("DELETE * FROM table");
auto updatedRows = conn.execute("UPDATE table SET quantity = 1 WHERE price > 100");
assert(conn.query("INSERT INTO table VALUES(1, 50)") == 1);

    Returns

    Number of affected rows.
