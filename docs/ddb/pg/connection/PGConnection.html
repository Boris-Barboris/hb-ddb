    Class representing connection to PostgreSQL server.

    Inherits from

    • Object (base class)

    Constructors

    Name Description
    this Opens connection to server.

    Methods

    Name Description
    begin Starts a transaction.
    close Closes current connection to the server.
    commit Commits a transaction.
    execute Executes a non query command, i.e. query which doesn't return any rows. Commonly used with data manipulation commands, such as INSERT, UPDATE and DELETE.
    executeRow Executes query and returns only first row of the result.
    executeScalar Executes query returning exactly one row and field. By default, returns Variant type.
    listen Subscribes to a channel.
    query Executes query which returns row sets, such as SELECT command.
    rollback Rolls a transaction back.
    transaction Work in a transaction context.
    transaction Work in a transaction context.
    unlisten Unsubscribes from a channel.
