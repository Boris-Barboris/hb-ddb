    Struct PGField

    Contains information about fields returned by the server

    Fields

    Name Type Description
    binaryMode bool Whether the response from the server is in binary (or text otherwise)
    index short If the field can be identified as a column of a specific table, the attribute number of the column; otherwise zero.
    modifier int The type modifier (see pg_attribute.atttypmod). The meaning of the modifier is type-specific.
    name string The field name.
    oid uint The object ID of the field's data type.
    tableOid uint If the field can be identified as a column of a specific table, the object ID of the table; otherwise zero.
    typlen short The data type size (see pg_type.typlen). Note that negative values denote variable-width types.
